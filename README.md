# Workflow repo for the CA


## 📦 Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
npm install
```
🛠️ Development

```bash
npm run dev
```

🧪 Testing
Playwright
```bash
npm run test
```
Vitest
```bash
npx vitest run
```
