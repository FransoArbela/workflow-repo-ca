# Workflow repo for the CA


## 📦 Installation

Clone the repo and install dependencies:

```bash
gh repo clone FransoArbela/workflow-repo-ca
cd workflow-repo-ca
npm install
```
## 🛠️ Development

```bash
npm run dev
```

## 🧪 Testing
Playwright
```bash
 npx playwright test
```
Vitest
```bash
npx vitest run
```
